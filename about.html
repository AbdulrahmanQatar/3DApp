About This Project

Project Overview

This 3D application was created as part of an assignment for the 3D Applications module. The project demonstrates the integration of 3D models created in Blender with web technologies including HTML5, CSS3, and JavaScript (Three.js).

Technical Implementation

The application features three detailed 3D beverage models created in Blender. These models have been exported as GLB files and imported into a Three.js scene for web display. The user interface provides multiple ways to interact with these models:

  • Model selection through an intuitive button interface
  • Wireframe mode toggle to examine the geometric structure
  • Lighting controls to adjust the visual presentation
  • Camera position controls for different viewing angles
  • Auto-rotation toggle for continuous model viewing

Responsive Design

The application features a fully responsive design that works on devices of all sizes, from desktop computers to mobile phones. The layout adjusts dynamically to provide the best possible user experience regardless of screen size.

Model Creation Process

All 3D models were created using Blender 3.6. The modeling process involved:

  • Creating basic shapes using primitives
  • Applying modifiers to achieve complex geometries
  • UV unwrapping for proper texture application
  • Material creation with PBR principles
  • Optimization for web performance
